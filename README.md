# C++ Fundamentals Practice

Small C++ programs practicing core programming fundamentals — variables, loops, conditionals.

- **even_odd.cpp** — checks if a user-entered number is even or odd
- **simple_interest.cpp** — calculates simple interest and total amount from principal, rate, and time
- **login_simulator.cpp** — simulates a login system with username/password comparison
- **multiplication_table.cpp** — prints the multiplication table (1–10) for a user-entered number
